【新聞】亞洲首檔雲公司 伊雲谷今登興櫃

最新資訊 2018 / 07 / 17

由台新證券輔導的伊雲谷（6689）是一間以「雲」而誕生的新創公司，今（17）日登錄興櫃交易，成為亞洲第一檔「雲端數位管理服務公司」。

台新證券表示，伊雲谷專注在 AWS 雲端服務，以 AWS 公有雲底層基礎架構，扮演中間服務商角色，就各種資料搬遷、設定、開發、優化及運維等服務，提供企業客戶高信賴的維運服務。

伊雲谷提供 AWS（Amazon Web Service）專業服務，分別於 2016 年 12 月取得 AWS 受管服務合作夥伴、2017 年 4 月取得 AWS 原廠授權培訓夥伴，並提供完整且正規的 AWS 教育訓練，以及同年4月成為大中華區第一間 AWS Premier Consulting Partner，並於 11 月通過 ISO 27001 與 ISO 27017 專業驗證，是經過 AWS 與第三方稽核與認可的雲端專家。

據了解，全球許多知名雲服務提供商，如 Amazon 及 Google，皆已申請 ISO 27017 驗證並取得證書，來進一步證明其雲服務安全符合國際標準水準。由此可知，獲得此二項驗證，除了證明伊雲谷符合雲端專業能力的標準，也可供企業用戶評估、採用該公司技術的重要依據。

除此之外，伊雲谷在「行銷&商務」、「行動裝置」領域展現出熟練技術，在 2017 年通過 AWS Mobile Competency 及 AWS Marketing and Commerce Competency，又於 2018 年 2 月通過 AWS DevOps Competency，提供解決方案。

伊雲谷近兩年的營運成長，如雲端產業及大數據運算呈現高速成長，2016 年至 2017 年雲端服務營收倍數成長，倍數的營收成長除反映伊雲谷處於雲端及大數據未來 10 年成長性最高的產業外，並代表營運策略正確。伊雲谷提供AWS的附加服務，將持續帶來高毛利及高獲利，除深耕台灣市場外，將觸角伸至中國及東南亞，為營運注入新的成長動力。
