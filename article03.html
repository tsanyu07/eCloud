用人工智慧駕馭你的 AWS 伊雲谷推出雲平台 Atlas

最新資訊 2017 / 08 / 29

對於不少 IT 人員或工程師來說，光是平日要維護企業的雲端資源，就已經是個令人費心的議題，更別說要思考如何更有效率的運用這些資源，甚至是要降低雲端的使用成本。

為了解決這樣的問題，伊雲谷數位科技(eCloudvalley)發表了以人工智慧為基礎的新產品 — Atlas 雲端管理平台，讓企業不再煩惱如何管控雲端環境。

伊雲谷為 AWS 亞馬遜雲端服務在大中華區最具規模的專業服務夥伴，目前集中於經營雲端服務託管(MSP)、顧問、教育訓練、系統整合，以及各種解決方案在雲端的開發部署，其中 MSP 服務即提供客戶雲端環境的專業級維運管理與技術支援。

奠基於對 AWS 技術的最佳掌握度以及純熟的 MSP 能力基礎上，伊雲谷進一步發表了雲端管理平台 Atlas，其核心的人工智慧自動化維運技術，不僅能讓 AWS 用戶在分秒之間快速部署所有雲端資源，還能自動將這些資源進行最佳化，強化雲端服務效能並降低維運成本。

除此之外，平台也提供視覺化的使用分析報告與安全檢測服務，讓 IT 人員或管理階層能更直覺、更快速地掌握服務狀況，並以此作為擬定企業 IT 策略的有效決策依據。

而對企業來說，如何應對 IT 相關的突發狀況亦不容輕易忽視，因此，伊雲谷的 Atlas 平台也針對企業可能面對的風險引入了人工智慧的最新技術，系統將根據用戶在 AWS 上過去的使用狀況進行大數據分析與機器學習，在自動偵測到 AWS 雲端資源處在異常使用狀態時(如：非預期內的過高成本)，即會迅速對管理者發布警告通知，對突發狀況進行最及時的處理，將影響降至最低。

另外，平台所提供的一鍵快速備份與排程功能，也更確保企業在資料與雲端資源的完整性，進一步完善了企業面對意外時的處理能力。

在雲端與人工智慧的這波潮流下，如何透過新科技促進企業的數位轉型，是每個企業專業經理人必須面對的重要課題。伊雲谷致力於透過全面且專業的產品與服務，讓客戶快速地邁向成功，而這次雲端管理平台 Atlas 的推出，也再次宣告著伊雲谷協助客戶走在產業的尖端，提升 IT 管理並拓展商業機會的決心。
